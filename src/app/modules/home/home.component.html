توضیحات مربوط به پروژه:

پروژه پیش رو تحت عنوان initial جهت استفاده به عنوان سورس پایه با ویژگی های زیر تولید و بهره برداری شده است:

  • رعایت معماری و فولدربندی استاندارد
  • استفاده از Variableهای مشخص جهت مدیریت رنگ بندی و فواصل
  • استفاده از کلاس‌های شخصی سازی شده جهت مدیریت buttonها، badgeها، dropdownها، datepickerها، فیلدهای فرم شامل inputها، checkboxها، radioButtonها، fileInputها و ...
  • استفاده از packageهای استاندارد جهت مدیریت پیغام‌‌ها، datepicker شمسی و ...
  • استفاده از bootstrap5.3 جهت مدیریت responsive و سایر قابلیت‌های آن
  • استفاده از bootstrap icons جهت مدیریت آیکن ها
  • ایجاد کامپوننت شخصی سازی شده جهت بهره بردازی از آیکن‌ها در فرمت svg مدرن تعریف شده در پروژه
  • ایجاد کامپوننت و سرویس مرتبط جهت شبیه سازی قابلیت new tab مرورگرهایی مانند chrome
  • ایجاد کامپوننت شخصی سازی شده جهت مدیریت راست کلیک (context menu)
  • ایجاد کامپوننت تقویم شمس قابل شخصی سازی

به جهت راحتی استفاده از کلاس‌ها، سرویس‌ها و کامپوننت‌های ایجاد شده، نمونه کدهای مرتبط در این صفحه قرار گرفته است.
